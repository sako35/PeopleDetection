# PeopleDetectionRTC
### 概要
- [Kinect v2センサ](https://developer.microsoft.com/ja-jp/windows/kinect)を用いて人物検出を行うことができるRTCです。
- Kinect v2センサの視野内にいる人数および、人物それぞれの位置を推定し出力します。
